概述

Markdown是一种书写的格式，它的目标是称为一种适合网络的书写语言。Markdown兼容HTML，不在markdown范围内的标签都可以直接在html里面撰写，直接加标签即可，除了<div>,<p>,<table>,<pre>需要在前后加上空行与其他段落进行区分，同时它们的开始和结尾标签不能用制表符或空格来缩进：

这是一个普通段落

<table>
  <tr>
    <td>foo</td>
  </tr>
</table>

这是一个普通段落

在html区块内的markdown语法不会被处理。但是HTML的区段（行内）标签如 <span> , <cite>, <del>可以在Markdown的段落、列表或标题里随意使用，也可以直接用HTML标签。Markdown语法在区段（行内）标签间是有效的。

在 HTML 文件中，有两个字符需要特殊处理： < 和 & 。 < 符号用于起始标签，& 符号则用于标记 HTML 实体，如果你只是想要显示这些字符的原型，你必须要使用实体的形式，像是 &lt; 和 &amp;。Markdown会自动转换html的实体。Markdown让你可以自然地书写字符，需要转换的由它来处理好了。如果你使用的 & 字符是 HTML 字符实体的一部分，它会保留原状，否则它会被转换成 &amp;。同样，如果 < 作为标签的一部分不会转换，否则会转换成 &lt;。不论是行内还是区块， < 和 & 两个符号都一定会被转换成 HTML 实体。

段落和换行

一个markdown段落的前后要有一个以上的空行，普通段落不允许用缩进。同时一个markdown段落是由一个或多个连续的文本行组成的，因此一个Markdown段落允许有多行，但是其并不会将每一个换行符都转换为 <br />，因此即使在一个段落中有多行，显示的时候仍是在一行上。如果要确实要强制 Markdown 插入 <br />，就在插入处输入两个以上的空格。

标题

在行首插入1到6个#，对应标题1到标题6，行尾无所谓，可以闭合，也可以不闭合

区块引用

在行首添加 >

> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
> consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
> Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.
> 
> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
> id sem consectetuer libero luctus adipiscing.

当然也可以偷懒只在每一个段落的开头添加>。可以嵌套，只要加上不同数量的>即可。

列表

Markdown支持有序列表和无序力表。无序列表使用*,+,-作为列表标记。有序列表使用数字作为标记。列表标记放在最左边，标记后面一定要跟上一个以上的空格或制表符：

* This is first
* This is second

当然，列表项目也支持多段落，但是每个项目下的每个段落都必须用4个空格或一个制表符缩进。如果项目内放进区块引用，>就需要缩进。如果项目内含有代码区块，该代码区块就需要缩进两次，即8个空格或2个制表符。

代码区块

代码区块需要缩进4个空格或1个制表符，显示时这个缩进会被移除并转换为<pre><code>

分割线

可以在一行内输入三个以上的*或-，可以在之间加入空格，不能有其他东西。

链接

Markdown支持两种格式的链接：行内式和参考式。

行内式：
This is [example](http://example.com)

参考式：
This is [example][id]
接着在文章的任意位置将链接定义的内容定义出来：
[id]: http://www.example.com  #冒号后需有一个以上的空格或制表符

强调

Markdown使用*,_作为标记，被其包围的字词被转换为<em>，被两个*或_包围的会被转换为<strong>。

代码

如果要标记一小段代码，可以用反引号（`）包起来

图片

Markdown使用和链接很相似的语法来标记图片，也分为行内式和参考式。

行内式：
![example](/path/to/pic)
参考式：
![example][id]
[id]: /path/to/pic

其他

反斜杠\被用来转义

`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧

+   加号

.   英文句点
!   惊叹号
