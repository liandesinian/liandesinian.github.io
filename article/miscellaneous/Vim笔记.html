Vim笔记

i -> insert模式，按ESC退出到Normal模式
x -> 删除当前字符
:wq -> 存盘并退出
dd -> 删除当前行并将其保存到剪贴板中
:help <command> 帮助

各种插入模式

a -> 在光标后插入
o -> 在当前行后插入一新行
O -> 在当前行前插入一个新行
cw -> 替换从光标当前位置后到一个word结尾的字符, 并进入插入模式
ce -> 同cw
cb -> 替换从光标当前位置前到一个word开头的字符，并进入插入模式
cW、cE、cb -> 同前，只不过是一个WORD

光标移动

hjkl -> 移动光标
0 -> 数字0, 到行头
^ -> 到本行第一个不是空白字符的位置
$ -> 到本行行尾
g_ -> 到本行最后一个不是空白字符的位置
/pattern -> 搜索字符串
:N -> 到第N行
gg -> 到第一行
G -> 到最后一行

word: 包含连续的字母、数字和下划线或者连续的其他非空的符号，彼此之间以空白符隔开。

w: 移动到下一个word的开头
e：移动到下一个word的结尾
b：移动到前一个word的开头

WORD：包含连续的非空的字符，彼此之间以空白符隔开，比word的范围大。

W：移动到下一个WORD的开头
E：移动到下一个WORD的结尾
B：移动到前一个WORD的开头。

NOTE: 中文的话连续的字算是word，标点符号算是下一个word。而连在一起中间无空白符的算是WORD，包含中文、字符、标点。

%: 匹配括号移动，需先移动到括号上
\>和#：匹配光标当前所在的单词，移动光标到下一个（或上一个）单词

光标移动命令很重要，很多命令都可以和光标移动命令联动，如下：

<start position><command><end position>

在当前行上移动光标：0 ^ $ f F t T

f<character> -> 到下一个<character>字符处
t<character> -> 到下一个<character>前的第一个字符
F、T -> 和f, t相同，只不过方向相反

Note: 可以和d, y, v等结合，eg：dta -* 表示从当前位置删除内容，直到遇到字符a。

区域选择

<action>a<object> 和 <action>i<object>。选择<object>之间的内容

 <action>可以是任何的命令，若d, y, v等
 <object>可能是w一个单词，W一个以空格符分隔的单词，s一个句子，p一个段落，也可以是一个特别的字符：", ', (), {}, []
 a表示会包含<object>，i不包含<object>

块操作

 <ctrl-v> 典型操作：

 ^ -> 移动到行头
 <ctrl-v> -> 选择块
 hjkl选择区域
 I-- <ESC> -> 大写I，插入"--", 按ESC生效

拷贝粘帖

 p/P -> 粘帖。p表示在当前位置之后，P表示在当前位置之前
 yy -> 拷贝当前行

撤销

 u -> 撤销
 CTRL-r -> redo

文件操作

 :e <filename> -> 打开文件
 :w  -> 保存文件
 :saveas <filename> ->另存为
 :q! -> 退出不保存
 :bn -> 可以同时打开多个文件，用此命令切换文件

重复

 . -> 重复上一次命令
 N<command> -> 重复命令N次

自动补全：<ctrl-n> <ctrl-p>

在插入模式下，输入一个词的开头，按<ctrl-p>或是<ctrl-n>自动补全（需是前面出现过的）

宏录制：qa操作序列，@a，@@

 qa -> 把操作记录在寄存器a上
 @a -> replay录制的宏
 @@ -> replay最新的宏

可视化选择:v, V, <ctrl-v>

一旦选择好后，可以做下面的事情：

 J -> 把所有的行连接成一行
 <或> -> 左右缩进
 = -> 自动缩进

分屏：sp

 :sp -> 分屏
 <ctrl-w><dir> -> <dir>是方向，用来切换分屏
 <ctrl-w>_：最大化尺寸
 <ctrl-w>+或<ctrl-w>-：增大（缩小）尺寸
